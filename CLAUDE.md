# Manu — VPA Petition Desk

## Design System
Always read `DESIGN.md` before making any visual or UI decision.
All font choices, colors, spacing, icons, and aesthetic direction are defined there.
Do not deviate without explicit user approval.
In QA/review, flag any UI code that does not match `DESIGN.md`.

- **Direction:** Civic Slate (efficient and serious, cool digital-government).
- **Type:** Fraunces + Noto Serif Tamil for headlines/long reading, Catamaran (Latin + Tamil)
  for UI/body, IBM Plex Mono for all numbers (tabular: counts, dates, amounts, tokens, IDs).
- **Color:** Slate Indigo `#21395B` primary, Signal Blue `#2F6FED` accent (one key action),
  cool neutrals, semantic urgency scale (critical/high/medium/low).
- **Icons:** lucide-react, 1.75px stroke.
- **Surfaces:** citizen intake (roomy, Tamil-first), PA portal (dense), display board (oversized).
